Графическая модификация ENBSeries v0.132 для TES Skyrim

В этой версии исключительно исправления известных ошибок предыдущих.


ВНИМАНИЕ!
bFloatPointRenderTarget=1 должно быть выставлено в SKyrimPrefs.ini для работы мода.
Запустите SkyrimLauncher.exe, чтоб перенастроить видеоопции игры.



Версия 0.131:
Исправлен туман для воды, добавлена категория параметров [PARTICLE] для контроля
различных прозрачных объектов (в том числе водопадов).

Версия 0.130:
Добавлено сглаживание Edge AA, поскольку все версии свыше 0.119 не поддерживают
аппаратный антиалиазинг MSAA из-за ограничений Dx9. У некоторых пользователей
проявляются дефекты из-за кода 0.129 beta, но тестеров нет и поэтому не исправлено.

Версия 0.126:
Встроен редактор конфигурации (GUI) который можете вызвать одновременным нажатием
кнопок SHIFT и ENTER. Некоторые настройки требуют перезагрузки игры и не редактируемы.
0.126 версия отличается от 0.125 дополнительными параметрами ssao для интерьера;
иным кодом ограничителя кадров (некоторые пользователи жаловались); интерфейс теперь
можно редактировать вводом данных клавиатурой.

Версия 0.125:
Встроен интерфейс (GUI) для редактирования настроек внутри игры.

Версия 12.12.12:
Оптимизировано отражение. Исправлены некоторые артефакты предыдущей версии.

Версия 0.123:
Добавлено отражение для тестирования его надобности для TES Skyrim, а производительность
эффекта и качество пока не играбельны для большинства. Исправлена ошибка при значении AOMixingType=2.

Версия 0.122:
Улучшен код исправления для параллакса (смотрите parallax.txt). Изменено качество ssao/ssil
эффектов и методы их смешивания со сценой (добавлены новые параметры). Пока еще не поборол
артефакт светлого контура вокруг объектов при некоторых условиях, скоро обновлю. Отражения
все еще не доделаны. Ну и главное, что производительность уже от 0.121 версии, кроме ssao.

Версия 0.121:
Переработка качества эффектов и производительности.

Версия 0.119:
Добавлены параметры interior для настройки внутри помещений отдельно от *Day, *Night.
В шейдеры добавлена глобальная переменная EInteriorFactor для переключения между
внутренними и внешними локациями.

Версия 0.117:
Для обхода проблемы ярких волос добавлен параметр GammaCurve, который аналогичен параметрам
ColorPowDay и ColorPowNight, но работает для всего изображения в целом и предназначен
для их замены. Однако использование глобального значения может потребовать перенастройки
прочих параметров. На всякий случай добавил глобальный параметр Brightness, который тоже
на все изображение применяется.
Добавлена подсветка облаков как в версии для Fallout New Vegas или GTA 4. Параметры
эффекта CloudsEdgeClamp и CloudsEdgeIntensity может понадобится скорректировать для каждой
настройки мода.
Эта версия содержит исправление ошибок предыдущих и новый программируемый эффект
для солнца, позволяющий реализовать линзовые блики, например.
Для этого используется файл enbsunsprite.fx и текстура для него enbsunsprite.tga,
enbsunsprite.png или enbsunsprite.bmp. Фактически, обычному игроку пользы сейчас
никакой от этого эффекта, он предназначен для моддеров, которые умеют редактировать
шейдеры и создадут собственные вариации. Вероятно позже сделаю сам, но пока еще
много более важных дел (depth of field эффект тоже ведь все еще в качестве примера
выполнен).

Версия 0.114:
Отличается от 0.113 эффектом sky lighting (ambient occlusion). Для наилучшего
качества этому эффекту необходимы объекты, рисуемые в тень, поэтому вручную отредактируйте
данные строки в файле SkyrimPrefs.ini:
bTreesReceiveShadows=1
bDrawLandShadows=1
bShadowsOnGrass=1
И не забудьте поставить bFloatPointRenderTarget=1 там же.
Новые параметры:
UseOriginalObjectsProcessing при значении true отключаются все изменения 3д объектов,
что может быть полезно тем пользователям, которым не нужна вся функциональность мода,
например лишь для исправления параллакса.
FixGameBugs предназначен для исправления ошибок 1.5.26.0 версии игры для ATI пользователей.
FixParallaxBugs исправление ошибок игры при работе с модификацией, добавляющей параллакс,
временно работает только на видеокартах NVidia.
AntiBSOD и SpeedHack немного повышают производительность, UseComplexIndirectLighting
влияет на скорость только при UseIndirectLighting=true, качество при этом ухудшается.
EnableDetailedShadows временно работает только на видеокартах NVidia, новый эффект для
повышения качества теней. (пример видео http://youtu.be/9uG-s9cuPPM)
ShadowCastersFix позволяет тени от гор или других больших объектов отбрасывать, но не для
всех.
ShadowQualityFix временно работает только на видеокартах NVidia и за счет небольшого
снижения производительности значительно снижает шум от теней.
DetailedShadowQuality временно работает только на видеокартах NVidia, значения качества
сильно влияют на производительность нового эффекта детализированных теней.
UseBilateralShadowFilter временно работает только на видеокартах NVidia, позволяет улучшить
качество теней и частично устранить контуры.

Версия 0.113:
Отличие в эффекте лучей от солнца.
Эта версия похожа на 0.103 с новым эффектом теней, исправлениями для параллакса,
оптимизацией. В предыдущей пользователям ATI карт надо было выставлять значение
ForceFakeVideocard=true для правильной работы некоторых эффектов, но в этой попытался
сделать ту же оптимизацию, что и для NVidia карт, хотя не знаю, что из этого получилось,
не на чем тестировать. Заодно исправление параллакса для http://skyrim.nexusmods.com/downloads/file.php?id=16919
под ATI карты тоже должно работать. Немного переделан код смешивания SSAO и Indirect Lighting
для повышения качества, Изменен метод просчета факторов для контроля дальности отрисовки
SSAO эффекта (FadeFogRangeDay и FadeFogRangeNight). Изменил режимы качества для детальных
теней (качество -1 это экстремальный режим, 0 - высокий, 1 - средний, 2 - низкий) с
для повышения детализации.

Версия 0.110:
Является копией 0.103 с новым эффектом теней, исправлениями для параллакса,
оптимизацией. Наибольшая производительность на данный на видеокартах NVidia, пока не
представляется возможности разрабатывать на ATI, ждите обновлений или выставляйте
параметр ForceFakeVideocard=true.
Обратите внимание, что в отличие от 0.103, этот мод не создает фейковую видеокарту "ENB"
на видеокартах NVidia, поэтому придется заново настроить графические опции игры, если
ранее использовали мод 0.103 или более ранний. Для пользователей ATI остается старый
режим, который переключается новым параметром в файле конфигурации ForceFakeVideocard.

Версия 0.109:
Тестовая версия оптимизаций и исправлений под патч игры 1.5.26.0.

Версия 0.108:
Упрощенная версия без многих эффектов, но с высокой производительностью.

Версия 0.105, 0.106:
Оптимизация кода для снижения нагрузки на процессор.

Версия 0.103:
Реализован экспериментальный код инжектора вместо обычного d3d9 враппера. Это может
помочь пользователям с ноутбуками Optimus технологии и тем, кто использует оверлейные
утилиты вроде EVGA, Afterburn, D3D Overrider, XFire и им подобные. Графические изменения
это возврат к SSAO из версии 0.099 с изменением интенсивност иindirect lighting.

Версия 0.102 Tatsudoshi:
Исправление мелких ошибок, в основном с этой версией производилась оптимизация, пока
еще частичная. Многие известные баги остались, исправлю позже.

Версия 0.101:
Исправлены ошибки предыдущей версии, которые у меня были видны. Добавлен код
программируемого эффекта depth of field (но не завершенный, потом буду разбираться).
В новом файле enbeffectprepass вы можете сделать не только depth of field, он
поддерживает многопроходный рендеринг и выполняется до enbeffect.fx, работая с
hdr изображением, в отличие от effect.fx. Изменен стандарт для внешних шейдеров,
избавился от ScreenSize и ScreenScaleY, заменив на один ScreenSize вектор из 4-х
значений. Старые файлы эффектов по этой причине могут не работать, надо заменить
по крайней мере ScreenScaleY на ScreenSize.z и float ScreenSize; замените на
float4 ScreenSize;. Переделал блум, отказавшись от параметров радиуса и повысив
качество (только формулы смешивания еще не затрагивал).

Версия 0.100:
Удален параметр CyclicConfigReading (каждые 5 секунд чтение конфигурации), отныне
загрузка значений происходит нажатием кнопки BACK (можно заменить KeyReadConfig).
Добавлен почти весь код из моего патча AntiFREEZE TES Skyrim 0.096, в том числе и
большинство параметров файла конфигурации. FPS limiter теперь тоже в наличии. Вернул
функцию захвата скриншотов (только в полноэкранном режиме) и счетчик кадров.
SSAO эффект сделал еще одной версии, упрощенный и более быстрый, для этого выставьте
в файле enbseries.ini параметр UseIndirectLighting=false и перезапустите игру.
Внедрил параметр контроля дальности эффекта SSAO в зависимости от тумана. Немного
повысил производительность SSAO. Разделил освещение на день и ночь. Параметры
адаптации в конфиге привел в порядок, но все еще в шейдере enbeffect.fx стоят
ограничители, можно вручную затереть (текст). Включены параметры SubSurfaceScattering
для снижения чрезмерной светимости персонажей в тенях и ярких тонких полос (плохая
реализация разработчиками игры). Добавлен параметр уровня светимости окон, пока что
он влияет на некоторые эффекты, например заморозку. Добавлен параметр ShadowObjectsFix
для включения более правильных теней от гор. Исправлены разные ошибки. SSAO теперь
работает с антиалиазингом.

Версия 0.099:
Исправлен вылет игры в вечернее время суток. Отключен захват скриншотов, который
по непонятной причине приводил у некоторых пользователей к непрерывному их созданию.
Активирован код палитровых текстур (enbpalette.bmp, png, tga files). По умолчанию
отключен эффект SSAO, редактируйте UseAmbientOcclusion=true в enbseries.ini и
убедитесь, что не включен антиалиазинг в игре или в драйверах. Не тестирован с
другими d3d9.dll модификациями, скорее всего они запортят работу данного мода.

Версия 0.098:
Внедрен код из патча AntiFREEZE для исправления некоторых ошибок игры и увеличения
стабильности с этим модом. Добавлены параметры для управления цветом тумана.

Версия 0.097:
Меня уже достало править ошибки игры, чтоб "подружить" ее с модом, да еще
и официальные патчи будут выходить и сбивать весь прогресс, поэтому выпускаю
пока упрощенную версию. Вылеты игры версии 1.1 при использовании мода возникают
очень часто, по крайней мере на моем компьютере, это не ошибки модификации.
Аппаратное сглаживание (антиалиазинг/мультисэмплинг) не поддерживаются на данный
момент, поэтому чтоб работал SSAO, отключайте сглажку. Оптимизация отсутствует,
собственно как всегда в ранних версиях ENBSeries, хотите повыше производительность,
отключайте SSAO или снижайте его качество. В игре есть некоторые ньюансы, с которыми
еще не разобрался, например в помещениях (пещерах и пр.) прямой свет разработчики
зачем-то оставили, да еще и направление либо снизу вверх, либо сбоку, так что
если увеличить его интенсивность в enbseries.ini, только на солнце будет правильно.
Из простейших обманных маневров могу предложить не трогать его, а зато все остальные
параметры менять, ну и общую интенсивность увеличить до нужных значений в файле
enbeffect.fx (это шейдер пост обработки). Те, кто уже имеет опыт настройки версии
для GTA 4, наверняка легко разберутся, что менять. Что касается enbeffect.fx,
там тоже черт голову сломит. Значения SubSurfaceScattering не увеличивайте, в игре
на персонажах встречается глюк в виде тонкой полоски, исправлю когда прекратят
штамповать официальные патчи (и опять, это баг игры, а не мой).

УСТАНОВКА:
Извлечь файлы из архива в папку игры.
Игру необходимо перезапускать через лаунчер после установки патча, чтоб произвелась
автоматическая переинициализация настроек (после уже можно менять под свои потребности).


ПРОБЛЕМЫ:
Патч не совместим с запущенными программами XFire, Afterburn, EVGA и аналогичными,
поэтому игра может не запускаться. Также в случае проблем рекомендуется поставить
более старые или более новые драйверы видеокарты, притом новые не обязательно менее
глючные, для относительно старых видеокарт лучше подобрать подходящую стабильную и
старую версию драйверов, например NVidia forceware 196.21 прекрасно подходит для
видеокарт gf8500-9800.
